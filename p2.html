Introdução ao Firebase

O Firebase é uma plataforma poderosa para o desenvolvimento de aplicativos móveis e web, oferecendo uma ampla gama de serviços integrados na nuvem pelo Google.

Originalmente adquirida pelo Google em 2014, o Firebase evoluiu para se tornar uma solução abrangente para desenvolvedores, facilitando desde o desenvolvimento até a distribuição e análise de aplicativos.

Firebase

Um banco de dados NoSQL em tempo real que permite armazenar e sincronizar dados entre clientes.

Autentificação

Gerenciamento completo de autenticação de usuários, incluindo autenticação por e-mail/senha e redes sociais.

Armazenamento

Armazenamento de arquivos na nuvem, ideal para imagens, vídeos e outros conteúdos acessados pelos usuários.

Hosteando

Hospedagem de conteúdo estático e dinâmico diretamente nos servidores do Firebase, com SSL gratuito.
